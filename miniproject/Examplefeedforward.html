Feedforward model Example


Visualisation of feedforward MLP


Activation Function:

Sigmoid Tanh

No. of Output: 1

No. of Inputs: 2

No. of Hidden Layers: 1

No. of Hidden Neurons: 4

Epoch: 0


Here, It is an example of feedforward neural network which is attempting to solve XOR problems using supervised learning technique. On the left side is the visualisation of the neural network solving the XOR problems by predicting whether the output is black or white. In the square box it shows the value 0 for black and 1 for white. The neural network classifies black on the top left corner and bottom right corner and white on top right corner and bottom left corner . it shows how Multilayer layer network can solve non linear problems. On the right side is the visualisation of the actual network which solve the XOR problem which shows the architechure of the network ,the number of input nodes, number of hidden layers and number of output nodes. The Neural network takes a datasets feed it to the neuron and check whether the output get the correct answer or not, if it gets the correct answer it stops otherwise it adjust the weight and feed it again to the neuron.
